How to install:

1. Find your Cataclysm: Dark Days Ahead folder for example - "C:\Games\CDDA" (This is where I install my games)

2. Copy the folder(s) of the mod(s) you want to use into your - "CDDA\data\mods" folder.

3. Select 'replace', when asked what you should do with the duplicate files. 

4. Launch the game and wala all the mods reactivated
